# Practice agile Game Framework Prject #  

## sprint 1 ## 
what: 点击一下两个文本框内的数值就会互相攻击，先为0就认为输。

detail:
1. 两个文本框。可以填入一个整数
2. 一个开始按钮。可以点击。
3. 点击开始按钮时两个文本框内的值按照随机1000ms-1500ms的一次的速度进行减值。
4. 一直减少到值为0为止。
5. 每一次的减少的值是0-10

----------
## sprint 2 ## 
what: 让逻辑执行为异步，血量可配置。

detail:
1. 可以在配置文件ini中配置P1和P1的血量，CD时间，攻击值。
2. 此ini只支持‘name=value’方式。
3. 所有的测试都是在单独线程之内。
4. 将逻辑抽象到单独的类型中

## sprint3 ##
what: 角色有不同的攻击技能，并且可以独立配置。

detail:
1. 每个角色独立成类。
2. 为每个角色独立加载配置。
